# Red LAC AI - Website

Sitio web para la Red Latinoamericana de IA para el Desarrollo Humano Integral.

## Archivos

- `index.html` - Página principal del sitio
- `logos.html` - Opciones de diseño de logo

## Cómo subir a GitHub Pages

### Paso 1: Crear repositorio en GitHub
1. Ve a https://github.com/new
2. Nombre del repositorio: `red-lac-ai` (o el que prefieras)
3. Marca como **Public**
4. NO inicialices con README
5. Click en "Create repository"

### Paso 2: Subir archivos
Opción A - Por la interfaz web (más fácil):
1. En tu nuevo repositorio, click en "uploading an existing file"
2. Arrastra los archivos `index.html` y `logos.html`
3. Click en "Commit changes"

Opción B - Por línea de comandos (si tienes Git instalado):
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/TU-USUARIO/red-lac-ai.git
git push -u origin main
```

### Paso 3: Activar GitHub Pages
1. En tu repositorio, ve a **Settings** > **Pages**
2. En "Source", selecciona **main** branch
3. Click en "Save"
4. Espera 1-2 minutos

### Paso 4: Ver tu sitio
Tu sitio estará disponible en:
```
https://TU-USUARIO.github.io/red-lac-ai/
```

Para ver los logos:
```
https://TU-USUARIO.github.io/red-lac-ai/logos.html
```
